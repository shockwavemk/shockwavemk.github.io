# Team Presentations

## 📊 Available Presentations

### 1. **Shared Dev Tooling** (`index.html`)
Technical overview of the ascli CLI, Copilot agents, and skills library.
- Problem: Knowledge silos, outdated runbooks, duplicated work
- Solution: Unified CLI + agent/skill framework
- Benefits: 10x productivity, multi-project support, self-documenting

**Best for:** Technical audiences, teams implementing shared tooling

---

### 2. **Sven's Day** (`sven-story.html`)
A narrative journey showing the pain of manual operations vs. automated ascli workflow.

**The Story:**
- Martin (Java expert) goes on summer vacation
- Sven (DevOps engineer) becomes temporary replacement
- First customer issue hits: cart update failures in production
- Without ascli: 2.5 hours of searching docs, AWS setup, SQL debugging
- With ascli: 10 minutes to root cause and answer

**Scenes:**
1. The setup (Martin leaves, Sven takes over)
2. The problem (customer issue email arrives)
3. The painful journey (Confluence → git repos → AWS → Athena)
4. Pain points breakdown (hours of manual work)
5. ascli to the rescue (one command)
6. The moral of the story

**Best for:** Team alignment, management presentations, onboarding new team members
