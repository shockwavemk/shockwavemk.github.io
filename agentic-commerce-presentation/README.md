# Agentic Commerce POC Presentation

An immersive 3D presentation about the Agentic Commerce Proof of Concept, built with **impress.js**.

## Features

- **3D Perspective**: Slides float in 3D space with depth positioning
- **Grass-Green Racetrack Background**: Immersive depth effect with far-positioned background
- **German Voiceover Script**: Complete 10-minute presentation narrative
- **Interactive Navigation**: Use arrow keys or mouse to navigate slides

## View the Presentation

Visit: **https://shockwavemk.github.io/agentic-commerce-presentation/**

## Keyboard Controls

- **Arrow Keys / Mouse**: Navigate between slides
- **Home/End**: Jump to first/last slide
- **F**: Fullscreen mode
- **ESC**: Exit presentation

## Content Overview

1. **Slide 1 - Title**: Ein Satz. Ein Agent. Eine Bestellung.
2. **Slide 2 - Problem**: Checkout in einem Satz. Nicht sieben Seiten.
3. **Slide 3 - Idea**: Wo wir hinwollen.
4. **Slide 4 - Solution**: Das bestehende Backend. Unverändert. KI-fähig.
5. **Slide 5-6 - Videos**: Produktsuche & Kompletter Checkout
6. **Slide 7 - Feedback Loop**: Selbstverbessernde Feedback-Schleife
7. **Slide 8 - What's Included**: Auf dem Weg zur Produktion
8. **Slide 9 - Finale**: Der nächste Lap startet jetzt.

## Technical Stack

- **Framework**: impress.js (Prezi-like 3D presentations)
- **Language**: HTML5 + CSS3 + JavaScript
- **3D Effects**: CSS transforms, perspective, and depth positioning
- **Background**: SVG-based grass track with tiling pattern
